🐦 CA Bird Atlas

Designed to support the California Bird Atlas, powered by eBird / Cornell Lab of Ornithology.

How to use

Tap any block on the map to see its effort statistics. Enable location to automatically track your current block while in the field.

Enabling location on mobile

Safari (iOS)
Settings → Privacy & Security → Location Services → Safari → While Using. Also check Settings → Safari → Location → Allow.
Chrome (Android)
Tap the lock icon in the address bar → Permissions → Location → Allow. Or: Settings → Site Settings → Location → allow this site.
Firefox (Android)
Tap the lock icon → Edit Site Permissions → Location → Allow.
Samsung Internet
Tap the lock icon → Location → Allow while visiting.
All browsers
Your device-level Location must also be on: iOS Settings → Privacy & Security → Location Services; Android Settings → Location → On.
